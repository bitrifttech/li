# homebrew-li

Homebrew tap for the `li` CLI assistant.

## Installation

```bash
brew tap bitrifttech/homebrew-li
brew install li
```

## Usage

```bash
# Configure your Cerebras API key
li config --api-key YOUR_CEREBRAS_API_KEY

# Try it out
li 'list files in current directory'

# Get help
li --help
```
